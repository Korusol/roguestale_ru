# Roguetale — Редактор переводов

GUI-приложение для перевода игры **Rogue's Tale** с английского на русский.

## Возможности

- **Таблица строк** — редактирование переводов из `Russian.xml` / `Russian_orig.xml`
- **Диалоги** — перевод `title`/`response` в XML-диалогах NPC и квестов
- Автоматическая генерация XML с чексуммой (SHA1) — игра принимает файл
- Поддержка `@(width)` для переформатирования текста по ширине
- Поддержка `#pp#` — хак для пустых строк (предотвращает генерацию артиклей игрой)

## Установка

### Требования

- **Python 3.10+**
- **PySide6** (устанавливается автоматически при запуске через `run.bat`)

### Запуск

**Способ 1 — двойным щелчком:**

```
run.bat
```

Скрипт сам установит PySide6, если его нет.

**Способ 2 — вручную:**

```bash
pip install PySide6
python main.py
```

## Использование

### 1. Таблица строк

1. Откройте вкладку **«Таблица строк»**
2. Загрузите `Russian.xml` и соответствующий `Russian.xml.txt`
3. Редактируйте переводы в таблице — поиск/фильтр по столбцам
4. Нажмите **«Сохранить»** — TXT обновится, XML с чексуммой перезапишется

**Формат TXT-файла:**

```
# комментарий (пропускается)
Grammar."a" некоторый текст
Skills_DoubleEnchant."name" Двойное зачарование
Quests."quest_1".3 абзац 3
```

- `TagName."key"` — ключ для обычной строки
- `TagName."key".N` — ключ для абзаца (paragraph N)
- `@(ширина)` в начале строки — автоперенос текста до указанной ширины (CJK-ширина: `W` = 2)
- `#pp#` в тексте заменяется на пробел в XML

### 2. Диалоги

1. Откройте вкладку **«Диалоги»**
2. Загрузите папку `dialogues/` с XML-файлами
3. В левой панели выберите персонажа — справа таблица `title`/`response`
4. Редактируйте. **Новые переводы сохраняются автоматически** в `dialogues_translations.json`
5. Нажмите **«Экспортировать в dialogues_Rus/»** — сгенерирует XML с чексуммой

**Переменные** (пропускаются автоматически): `STR_INTRO_2`, `SET_ENEMY`, `LEAVE_DUNGEON`,
`finish_quest`, `----------`, `STR_BUY[...]` и т.д.

## CLI-скрипты

Проект сохранил оригинальные скрипты, можно использовать без GUI:

| Скрипт | Назначение |
|---|---|
| `export.py` | Извлекает строки из `Russian_orig.xml` в `Russian_orig.xml.txt` |
| `import.py` | Читает `Russian.xml` + `Russian.xml.txt`, пишет `Russian1.xml` |
| `_translate_dialogues.py` | Читает `dialogues/`, пишет переводы в `dialogues_Rus/` (45 файлов) |
| `_extract_dialogues.py` | Показывает все непереведённые строки из диалогов |

Все скрипты работают с жёстко заданными именами файлов.

```

Russian_orig.xml.txt    — экспорт оригиналов (референс)

  txt_loader.py         — чтение/запись txt

  checksum_utils.py     — SHA1 для строк и диалогов
  string_model.py       — модель таблицы (QAbstractTableModel)
  string_table_tab.py   — вкладка таблицы строк
  dialogue_loader.py    — загрузка диалоговых XML

```

## Чексумма (SHA1)

Игра проверяет SHA1 в корневом теге `<strings checksum="...">`
и в каждом `<dialogue checksum="...">`. Алгоритм:

1. Удалить существующий checksum
2. Заменить `&#10;` на `\n`
3. Закодировать не-ASCII символы как `&#N;`
4. Вычислить SHA1, вставить обратно
